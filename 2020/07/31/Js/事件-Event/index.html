1.дәӢд»¶еӨ„зҗҶеҷЁ

1.1 дәӢд»¶еӨ„зҗҶеҷЁеұһжҖ§пјҲDOM0зә§пјү

иҜҘеҮҪж•°еңЁе®ҡд№үж—¶пјҢеҸҜд»Ҙдј е…ҘдёҖдёӘ event еҪўејҸзҡ„еҸӮж•°пјҢиҜҘж–№ејҸзҡ„й—®йўҳеңЁдәҺдёҖж¬ЎеҸӘиғҪз»‘е®ҡдёҖдёӘгҖӮ

btn.onblur еҪ“дёҖдёӘе…ғзҙ еӨұеҺ»з„ҰзӮ№ж—¶дјҡи§ҰеҸ‘blurдәӢд»¶
btn.ondblclick еҪ“еүҚе…ғзҙ дёҠеҸҢеҮ»йј ж Үе·Ұй”®дјҡи§ҰеҸ‘dblclickдәӢд»¶

1.2 иЎҢеҶ…дәӢд»¶еӨ„зҗҶеҷЁ - иҜ·еӢҝдҪҝз”Ё

жҲ‘д»¬еә”иҜҘйҒҝе…ҚдҪҝз”Ёиҝҷз§Қж–№ејҸгҖӮеӣ дёәе®ғдјҡдҪҝж Үи®°ж•°йҮҸеҸҳеӨ§пјҢиҖҢеҸҜиҜ»жҖ§еҚҙиҫғе·®гҖӮ еҶ…е®№/з»“жһ„ е’Ң иЎҢдёәд№Ӣй—ҙжІЎжңүеҫҲеҘҪзҡ„еҲҶзҰ»пјҢдҪҝеҫ—еңЁеӨ„зҗҶbugж—¶йқһеёёеӣ°йҡҫгҖӮ

2.дәӢд»¶еҶ’жіЎеҸҠжҚ•иҺ·

еҪ“дёҖдёӘдәӢд»¶еҸ‘з”ҹеңЁе…·жңүзҲ¶е…ғзҙ зҡ„е…ғзҙ дёҠпјҢзҺ°д»ЈжөҸи§ҲеҷЁиҝҗиЎҢдёӨдёӘдёҚеҗҢзҡ„йҳ¶ж®өвҖ”вҖ”жҚ•иҺ·йҳ¶ж®өе’ҢеҶ’жіЎйҳ¶ж®өгҖӮ

2.1жҚ•иҺ·йҳ¶ж®ө

  • жөҸи§ҲеҷЁжЈҖжҹҘе…ғзҙ зҡ„жңҖеӨ–еұӮзҘ–е…Ҳ<html>пјҢжҳҜеҗҰеңЁжҚ•иҺ·йҳ¶ж®өдёӯжіЁеҶҢдәҶдёҖдёӘonclickдәӢд»¶еӨ„зҗҶзЁӢеәҸпјҢеҰӮжһңжҳҜпјҢеҲҷиҝҗиЎҢе®ғгҖӮ
  • з„¶еҗҺпјҢе®ғз§»еҠЁеҲ°<html>дёӯеҚ•еҮ»е…ғзҙ зҡ„дёӢдёҖдёӘзҘ–е…Ҳе…ғзҙ пјҢе№¶жү§иЎҢзӣёеҗҢзҡ„ж“ҚдҪңпјҢз„¶еҗҺжҳҜеҚ•еҮ»е…ғзҙ еҶҚдёӢдёҖдёӘзҘ–е…Ҳе…ғзҙ пјҢдҫқжӯӨзұ»жҺЁпјҢзӣҙеҲ°еҲ°иҫҫе®һйҷ…зӮ№еҮ»зҡ„е…ғзҙ гҖӮ

2.2еҶ’жіЎйҳ¶ж®ө

  • жөҸи§ҲеҷЁжЈҖжҹҘе®һйҷ…зӮ№еҮ»зҡ„е…ғзҙ жҳҜеҗҰеңЁеҶ’жіЎйҳ¶ж®өдёӯжіЁеҶҢдәҶдёҖдёӘonclickдәӢд»¶еӨ„зҗҶзЁӢеәҸпјҢеҰӮжһңжҳҜпјҢеҲҷиҝҗиЎҢе®ғ
  • з„¶еҗҺе®ғз§»еҠЁеҲ°дёӢдёҖдёӘзӣҙжҺҘзҡ„зҘ–е…Ҳе…ғзҙ пјҢе№¶еҒҡеҗҢж ·зҡ„дәӢжғ…пјҢз„¶еҗҺжҳҜдёӢдёҖдёӘпјҢзӯүзӯүпјҢзӣҙеҲ°е®ғеҲ°иҫҫ<html>е…ғзҙ гҖӮ

2.3з”ЁstopPropagation()дҝ®еӨҚй—®йўҳ

еҪ“еңЁдәӢд»¶еҜ№иұЎдёҠи°ғз”ЁиҜҘеҮҪж•°ж—¶пјҢе®ғеҸӘдјҡи®©еҪ“еүҚдәӢд»¶еӨ„зҗҶзЁӢеәҸиҝҗиЎҢпјҢдҪҶдәӢд»¶дёҚдјҡеңЁеҶ’жіЎй“ҫдёҠиҝӣдёҖжӯҘжү©еӨ§пјҢеӣ жӯӨе°ҶдёҚдјҡжңүжӣҙеӨҡдәӢд»¶еӨ„зҗҶеҷЁиў«иҝҗиЎҢ(дёҚдјҡеҗ‘дёҠеҶ’жіЎ)гҖӮ

3.дәӢд»¶еҜ№иұЎ event

eventпјҢevtжҲ–з®ҖеҚ•зҡ„eгҖӮ иҝҷиў«з§°дёәдәӢд»¶еҜ№иұЎпјҢе®ғиў«иҮӘеҠЁдј йҖ’з»ҷдәӢд»¶еӨ„зҗҶеҮҪж•°пјҢд»ҘжҸҗдҫӣйўқеӨ–зҡ„еҠҹиғҪе’ҢдҝЎжҒҜпјҢжҜ”еҰӮпјҡ

дәӢд»¶еҜ№иұЎ e зҡ„targetеұһжҖ§е§Ӣз»ҲжҳҜдәӢд»¶еҲҡеҲҡеҸ‘з”ҹзҡ„е…ғзҙ зҡ„еј•з”ЁгҖӮ

и·ЁжөҸи§ҲеҷЁзҡ„дәӢд»¶еҜ№иұЎ

иҷҪз„¶DOMе’ҢIEдёӯеҜ№иұЎдёҚеҗҢпјҢдҪҶжҳҜдёӨиҖ…eventдёӯзҡ„е…ЁйғЁдҝЎжҒҜе’Ңж–№жі•йғҪжҳҜзұ»дјјзҡ„еҸӘжҳҜе®һзҺ°ж–№ејҸдёҚеҗҢпјҢеҸҜд»Ҙз”ЁеүҚйқўжҸҗеҲ°иҝҮзҡ„EventUtilеҜ№иұЎжқҘжұӮеҗҢеӯҳејӮгҖӮ

4.дәӢд»¶е§”жүҳ

4.1 й—®йўҳеҲқзҺ°

з»ҷ100дёӘжҢүй’®з»‘е®ҡдәӢд»¶пјҢдј з»ҹзҡ„еҒҡжі•е°ұжҳҜпјҡйҖҡиҝҮDOMж“ҚдҪңиҺ·еҸ–жҜҸдёҖдёӘжҢүй’®е…ғзҙ еҲҶеҲ«з»‘е®ҡдәӢд»¶еӨ„зҗҶеҮҪж•°гҖӮ

дҪҶеңЁJavaScriptдёӯпјҢж·»еҠ еҲ°йЎөйқўдёҠзҡ„дәӢд»¶еӨ„зҗҶзЁӢеәҸж•°йҮҸе°ҶзӣҙжҺҘе…ізі»еҲ°йЎөйқўзҡ„ж•ҙдҪ“иҝҗиЎҢжҖ§иғҪпјҢеӣ дёәйңҖиҰҒдёҚж–ӯзҡ„дёҺdomиҠӮзӮ№иҝӣиЎҢдәӨдә’пјҢи®ҝй—®domзҡ„ж¬Ўж•°и¶ҠеӨҡпјҢеј•иө·жөҸи§ҲеҷЁйҮҚз»ҳдёҺйҮҚжҺ’зҡ„ж¬Ўж•°д№ҹе°ұи¶ҠеӨҡпјҢе°ұдјҡе»¶й•ҝж•ҙдёӘйЎөйқўзҡ„дәӨдә’е°ұз»Әж—¶й—ҙгҖӮ
